## Getting Started

First, run the development server:

```bash
npm run dev

# mock server
npm run mock

```

## information

MSW(Mock Service Worker)

- 서비스 워커(Service Worker)를 사용하여 네트워크 호출을 가로채는 API 모킹(mocking) 라이브러리이다.
  즉, 브라우저에서 마치 백엔드 API인 척하여 프론트엔드의 요청에 가짜 데이터를 응답해주는 녀석이다.

NEXT_PUBLIC

- 브라우저에서 접근가능한 환경변수

- useFormStatus
